1. Overview

  • Clusters data by trying to separate samples in n groups of equal variance
  • Highly dependent on the initialization of the centroids
  • Object : Minimize inertia(=within-cluster sum-of-squares)
    • Inertia makes the assumption that clusters are convex and isotropic
    • Responds poorly to elongated clusters, or manifolds with irregular shapes
  • Parameters : number of clusters

Algorithm

  1. Choose n initial centroids
  2. While the centroids do not move significantly
    1. assigns each sample to its nearest centroid
    2. creates new centroids by taking the mean value of all of the samples assigned to each previous centroid

2. Example 

import numpy as np
from sklearn.datasets import make_blobs
from matplotlib import pyplot as plt

SEED = 42

X, y = make_blobs(
    n_samples=2000,
    n_features=2,
    centers=4,
    shuffle=False,
    random_state=SEED
)

for _ in np.unique(y):
    ix = np.where(y == _)
    plt.scatter(X[ix, 0], X[ix, 1], label=f"{_} (n={len(ix[0])})")
plt.legend()
plt.show()

from sklearn.cluster import KMeans

kmeans = KMeans(n_clusters=4, random_state=SEED).fit(X)
pred = kmeans.predict(X)

for _ in np.unique(pred):
    ix = np.where(pred == _)
    plt.scatter(X[ix, 0], X[ix, 1], label=f"{_} (n={len(ix[0])})")
plt.scatter(kmeans.cluster_centers_[:, 0], kmeans.cluster_centers_[:, 1], label="centers", c='cyan')
plt.legend()
plt.show()
