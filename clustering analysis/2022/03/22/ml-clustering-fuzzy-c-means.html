[MachineLearning] Cluster Analysis - Fuzzy C Means

KNU AIR week2

Mar 22, 2022 • 1 min read

clustering analysis

Content creators: 이주형

Content reviewers:

1. Overview

  • A form of clustering in which each data point can belong to more than one cluster
  • In fuzzy clustering, data points can potentially belong to multiple clusters.
  • The fuzzy c-menas algorithm is very similar to the k-means algorithm.
  • Parameters : number of clusters

Algorithm

  1. Choose a number of clusters.
  2. Assign coefficients randomly to each data point for being in the clusters.
  3. While the coefficients' change between iterations is no more than the given threshold:
    1. Compute the centroid for each cluster.
    2. For each data point, compute its coefficients of being in the clusters.

2. Example 

import numpy as np
from sklearn.datasets import make_blobs
from matplotlib import pyplot as plt

SEED = 42

X, y = make_blobs(
    n_samples=2000,
    n_features=2,
    centers=4,
    shuffle=False,
    random_state=SEED
)

for _ in np.unique(y):
    ix = np.where(y == _)
    plt.scatter(X[ix, 0], X[ix, 1], label=f"{_} (n={len(ix[0])})")
plt.legend()
plt.show()

import skfuzzy as fuzz # scikit-fuzzy

c_means = fuzz.cmeans(X.T, c=4, m=2, error=1e-3, maxiter=300)

centers, coefs = c_means[0], c_means[1]
pred = np.argmax(coefs, axis=0)
for _ in np.unique(pred):
    ix = np.where(pred == _)
    plt.scatter(X[ix, 0], X[ix, 1], label=f"{_} (n={len(ix[0])})")
plt.scatter(centers[:, 0], centers[:, 1], label='centers', c='cyan')
plt.legend()
plt.show()
